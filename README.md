# suiheisen
This is Suiheisen, a game meant to be paired with my own custom macropad (SuiheisenPad) that features four keys.

## Backstory
You're a Korean salaryman and you escape your chaebol office building to just run and chase the horizon. You don't know where your legs are taking you, but you don't care as long as it's far away from your dreaded cubicle.

## Controls
- z -> dodge left
- x -> OK/enter
- c  -> duck
- v -> right
